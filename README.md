# Aplicaci-n-GEI-Colombia
Plataforma de Data Insights construida con Streamlit y pmdarima. Transforma el Inventario Nacional de GEI en inteligencia estratégica. Ofrece análisis comparativo interanual, tendencias históricas sectoriales y pronósticos SARIMA de las emisiones netas a futuro. Herramienta esencial para la planificación climática corporativa en Colombia.
